## Key Concepts

### Container Types

**1. Manual Layout Container**
- Full control over child item positions
- Move items anywhere (X, Y coordinates)
- Resize items freely
- Group controls for unified movement

**2. Auto Layout Containers**

**Horizontal Container:**
- Child items arranged left-to-right
- Primary axis: horizontal
- Secondary axis: vertical
- Alignment options: start, middle, end, space between

**Vertical Container:**
- Child items arranged top-to-bottom  
- Primary axis: vertical
- Secondary axis: horizontal
- Same alignment options

**Note:** Horizontal and Vertical containers are the same control with different default directions. Direction can be switched.

### Container Hierarchy
- Containers can contain other containers
- Nested containers for complex layouts
- Parent container controls child positioning
- Child items move with parent container

### Alignment Properties

**Primary Axis (Direction axis):**
- Start - items at beginning
- Middle - items centered
- End - items at end
- Space Between - even spacing

**Secondary Axis (Cross axis):**
- Start, Middle, End alignment

## Power Apps Evolution

**First Generation (2017-2020):**
- Simple apps with gallery + form + icons
- Few controls per screen
- No need for grouping/sections

**Second Generation (Container era):**
- Modern UI with defined sections
- Static, partially responsive, or fully responsive
- Well-organized content groups
- Professional app layouts
