You can see how the agent works with the MCP server and starts utilizing the tools:
1. List tables — This will list out all the tables in the environment that I have access to.

KEY POINTS:
- First connection: Requires user approval
- Agent workflow:
  1. List tables (discovery)

- Real-time: Live data from Dataverse
