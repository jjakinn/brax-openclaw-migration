CHUNK 8: CARD CONTROL IN GALLERY WITH RESPONSIVE WRAP COUNT
Source: powerapps-modern-card-control-raza | Timestamp: 07:30 - 09:00

My document library has different types of files: images, PDFs, Word docs and more.

For my gallery data, I'll connect it to my connected documents library. So the items property is my document library.

---

Now I'll edit this gallery and insert the modern card control.

Now I'm trying to make this responsive. The card control, I'll position it at the top left.

Width: I'll set it to `Parent.TemplateWidth`
Height: I will set it to `Parent.TemplateHeight`

---

The gallery has a wrap count property. One meaning one column. If I set it to two, it will try and fit the data in two columns and so and so forth.

---

I'm trying to build a responsive screen. So the wrap count I'll drive based upon my screen size.

My screen control is called screen 2. So get me the size of the screen. Now size dynamically changes depending upon the device that the user is on.

For a desktop it's screen size 4, tablet is 3, mobile is 1.

---

The gallery template padding set to 25. So my cards are now equidistantly spaced out.

---

KEY POINTS:
- Insert card control in gallery
- Card sizing: `Parent.TemplateWidth`, `Parent.TemplateHeight`
- Gallery `WrapCount`: Controls columns
- Responsive wrap count: Based on `Screen.Size`
  - Desktop: 4 columns
  - Tablet: 3 columns
  - Mobile: 1 column
- Formula: `Screen.Size` (returns 1-4)
- Template padding: 25 for spacing
