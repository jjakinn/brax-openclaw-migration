KEY POINTS:
- SharePoint delegable filter: `StartsWith`, not `Search`
- Formula: `Filter(DataSource, StartsWith(Column, TextInput.Text))`
- Text input naming: `txtTitle` for reference
- Pattern: Filter function wraps the sort logic
- Delegation: Long formula but fully delegable
