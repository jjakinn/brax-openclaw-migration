KEY POINTS:
- Initial issue: Filter only applies on focus out (click outside)
- TriggerOutput property: Controls when filter applies
- Options:
  - `FocusOut`: When user leaves field
  - `Delayed`: After typing pause (recommended)
- Better UX: Use Delayed for live filtering
- StartsWith behavior: Matches beginning of text only
