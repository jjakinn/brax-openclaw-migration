KEY POINTS:
- Sort syntax: `Sort(DataSource, Expression, Order)`
- Expressions in Sort: NOT delegable
- Delegation limit: 2,000 rows
- Delegation warning: Blue dot indicates non-delegable
- Use case: Small datasets only when using expressions
