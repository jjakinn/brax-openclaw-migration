KEY POINTS:
- Flexible width: Distributes available space among controls
- Problem: Controls shrink too small on small screens
- Solution: Set `MinimumWidth` property on each control
- Behavior:
  - Wide screen: Expands beyond minimum if space available
  - Small screen: Respects minimum width, doesn't shrink below
- Fixed width: Turn off flexible width, set specific width
- Examples: Item ID = 80, Total Value = 90
