KEY POINTS:
- Header styling:
  - Align: Start (horizontal), Middle (vertical)
  - Font weight: Bold
  - Font color: Theme-based `App.Theme.Colors.Primary`
- Theme changes: Dynamically updates app colors
- Problem: Header doesn't scroll with gallery
- Solution: Set header container `LayoutMinimumWidth` = Gallery `LayoutMinimumWidth`
- Result: Header and gallery scroll together horizontally
