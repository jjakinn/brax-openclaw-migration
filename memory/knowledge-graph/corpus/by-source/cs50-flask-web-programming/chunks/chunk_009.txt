# Chunk 009: Shopping Cart Example

Sessions enable shopping carts - data persists across page loads.

```python
@app.route("/cart", methods=["POST"])
def add_to_cart():
    book_id = request.form.get("id")
    
    if "cart" not in session:
        session["cart"] = []
    
    session["cart"].append(book_id)
    return redirect("/cart")

@app.route("/cart")
def view_cart():
    books = db.execute("SELECT * FROM books WHERE id IN (?)", session["cart"])
    return render_template("cart.html", books=books)
```

Each user gets separate session data. Data persists until session cleared or expired.
