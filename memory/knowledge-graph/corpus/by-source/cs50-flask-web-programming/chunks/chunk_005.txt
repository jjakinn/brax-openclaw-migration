# Chunk 005: Form Validation and Security

Server-side validation is critical - never trust client-side data. Users can modify HTML with developer tools.

Example validation:
```python
SPORTS = ["basketball", "soccer", "ultimate frisbee"]

@app.route("/register", methods=["POST"])
def register():
    name = request.form.get("name")
    sport = request.form.get("sport")
    
    if not name:
        return render_template("error.html", message="Missing name")
    if sport not in SPORTS:
        return render_template("error.html", message="Invalid sport")
```

Pass valid sports list to template for dropdown generation:
```python
return render_template("index.html", sports=SPORTS)
```

Template loop:
```html
<select name="sport">
  {% for sport in sports %}
  <option value="{{ sport }}">{{ sport }}</option>
  {% endfor %}
</select>
```
