KEY POINTS:
- Navigation item properties:
  - `key`: Unique identifier (e.g., "home", "tasks")
  - `title`: Display text (e.g., "Home", "My Tasks")
  - `screen`: Target screen object (or `Blank()` for actions)
  - `icon`: Icon name from modern icon suite
- Icons: Use modern button control to browse available icons
- Logout example: `screen: Blank()` for actions without navigation
