To create the navigation menu I'll go and insert a gallery control. Let's pick the blank flexible height gallery.

I'll adjust this to begin right below the user image.

---

We are trying to make this a responsive dynamic menu experience.

The height for this gallery, we'll set it as `Parent.Height - Self.Y`.

Now the Y property is how many pixels from the top this control is placed at. You subtract that from the height of the parent. That's why it takes the entire remaining height of the component.

---

This gallery I'll rename to `galMenu`.

The X property of this gallery set it as 0.

The width property will be `Parent.Width`.

---

KEY POINTS:
- User department: `Office365Users.MyProfileV2().department`
- Gallery type: Blank flexible height
- Gallery name: `galMenu`
- Positioning:
  - X: 0 (left aligned)
  - Width: `Parent.Width`
  - Height: `Parent.Height - Self.Y` (fills remaining space)
