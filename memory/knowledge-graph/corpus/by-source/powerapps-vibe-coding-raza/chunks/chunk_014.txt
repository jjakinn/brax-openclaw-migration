KEY POINTS:
- Mobile: QR code scan to run on mobile device
- Browser: Play button runs in browser
- Sharing: Click share → Select users/groups
- Group sharing: Supported if Entra ID security-enabled
- **Critical**: Users need Dataverse security roles for table access
- Custom security roles: Create in solution context
- Privilege levels: Read, Collaborate, Full access per table
- Row-level security: Built into Dataverse
- One-go sharing: Assign security role during share operation
