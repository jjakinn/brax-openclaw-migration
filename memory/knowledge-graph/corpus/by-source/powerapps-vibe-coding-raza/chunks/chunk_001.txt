Now what this ensures is that any new artifact that I build in this environment and in this context the artifacts that I will build through vibe.powerapps.com will all be stored in my preferred solution. So I've set this to yes. I'll create my solution.

- Vibe portal: vibe.powerapps.com

- Environment switching available if needed
