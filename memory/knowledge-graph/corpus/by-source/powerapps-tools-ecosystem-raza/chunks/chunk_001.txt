A range of tools for app creation from no code to low code to vibe coding to pro code. The Power Apps ecosystem spans this entire range.

---

For information workers, we have the app builder agent directly available inside M365 Copilot.

For makers, we have the Power App Studio experience. You can build canvas apps and model-driven apps.

For makers and developers, we have vibe coding options through the new Power Apps endpoint: vibe.powerapps.com.

For developers, we have the pro code experience which is available through powerapps.codeapps.

---

KEY POINTS:
- Power Apps ecosystem spans: No-code → Low-code → Vibe-code → Pro-code
- Four target audiences:
  1. Information workers: App Builder Agent in M365 Copilot
  2. Makers: Power Apps Studio (Canvas + Model-driven)
  3. Makers & Developers: Vibe coding (vibe.powerapps.com)
  4. Developers: Pro code (Code Apps)
