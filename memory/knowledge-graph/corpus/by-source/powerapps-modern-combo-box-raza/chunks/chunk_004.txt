Item label text. You're not limited to display just one property here. I could use the concatenate function to concatenate the title along with other properties coming from my data source.

I'll concatenate text hyphen and then I'll concatenate thisItem.ID.

---

KEY POINTS:
- Item label text supports formula expressions
- Can display multiple properties using `Concatenate()`
- Example: `Concatenate(thisItem.Title, " - ", thisItem.ID)`
- Concatenated display appears in:
  - Dropdown list
  - Search results
  - Selected items
