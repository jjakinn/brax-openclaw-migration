## Course Overview

CS50 is Harvard's introductory computer science course, covering:
- Computational thinking
- Programming fundamentals (C, Python)
- Computer systems (memory, data structures)
- Web development (HTML, CSS, JavaScript, Flask)
- Databases (SQL)
- Final projects

---

## Major Topics

### Weeks 0-2: Introduction & C Programming
- Scratch visual programming
- C syntax and compilation
- Variables, conditionals, loops
- Functions and recursion

### Weeks 3-5: Memory & Data Structures
- Pointers and memory addresses
- Arrays and strings
- Dynamic memory allocation
- Linked lists, hash tables, tries

### Weeks 6-7: Python & SQL
- Python syntax and libraries
- File I/O and regular expressions
- Database design
- SQL queries and optimization

### Weeks 8-10: Web Development
- HTML, CSS, JavaScript
- Flask web framework
- APIs and JSON
- Sessions and authentication

3. **Debugging** - Finding and fixing errors
4. **System design** - Architecture decisions
5. **Problem solving** - Applying CS to real problems
