# CS50: Web Programming with Flask - Executive Summary

**Source:** Harvard CS50 Lecture  
**Instructor:** David J. Malan  
**Duration:** ~2.5 hours  
**Domain:** Web Development, Python, Flask

---

## One-Sentence Summary

A comprehensive introduction to web application development using Python's Flask framework, covering HTTP fundamentals, routing, templates, form handling, database integration, sessions, and REST APIs.

---

## Key Topics Covered

1. **HTTP & Web Servers** - Understanding GET/POST, routes, and request/response cycle
2. **Flask Framework** - Minimal Python web framework for building applications
3. **Templates (Jinja)** - Separating HTML from Python logic
4. **Forms & Validation** - Handling user input securely
5. **Databases** - SQL integration with SQLite
6. **Sessions** - User authentication and state management
7. **APIs** - Building and consuming RESTful services

---

## Core Concepts

- **Routes:** URL patterns mapped to Python functions
- **MVC Pattern:** Model-View-Controller architecture
- **GET vs POST:** HTTP methods for data retrieval vs submission
- **Templates:** Reusable HTML with dynamic content
- **Sessions:** Persistent user state via cookies
- **SQL Injection Prevention:** Using parameterized queries

---

## Practical Examples

- Hello World web app
- Registration forms with validation
- Database-backed applications
- Shopping cart with sessions
- REST API endpoints

---

## Prerequisites

- Python fundamentals
- HTML/CSS basics
- SQL knowledge (covered in prior lectures)

---

## Next Steps

Apply these concepts to build full-stack web applications with user authentication, database persistence, and API endpoints.
