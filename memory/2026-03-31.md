# Power Apps Code Apps Development — Texas Title Premium Calculator

## Project Completed

**App Name:** Texan Title Premium Calculator  
**Type:** Power Apps Code App (React + TypeScript)  
**Deployed To:** IT Help Desk environment (Power Platform)  
**Status:** ✅ Live and functional

## Key Technical Achievements

### 1. Power Apps Code Apps Workflow Established
- Learned complete CLI workflow from YouTube tutorial (Raza)
- Mastered `pac code init` and `pac code push` commands
- Successfully deployed React app to Power Platform

### 2. Critical Fixes Discovered

**White Screen Fix:**
```typescript
// MUST use relative paths in vite.config.ts
export default defineConfig({
  base: './',  // ← REQUIRED for Power Apps!
  plugins: [react()],
})
```

**Logo/Image Fix:**
- Power Apps blocks external image URLs
- Embedded base64 data URIs work BUT must be COMPLETE (not truncated)
- Resized logo to ~6KB for optimal loading

**Currency Input Fix:**
- Immediate formatting causes typing issues
- Solution: Raw input during typing, parse on blur
- Allows commas and decimals naturally

### 3. TDI-Compliant Calculator Features
- Texas promulgated rates (March 2026 effective)
- Simultaneous Issue logic (R-5)
- Refinance credits (R-8) with date-based tiers
- Endorsement calculations with minimums
- TRID disclosure calculations
- Full audit trail with rule references

### 4. UI Improvements
- Toggle for Optional Endorsements (collapsible)
- Changed to "Title Premium Calculator" branding
- Added company logo (base64 embedded)
- Clean, professional styling
