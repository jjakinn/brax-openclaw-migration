### What It Is
A browser-based multiplayer-style virtual world game built in Phaser 3. Players create accounts, design characters, walk around rooms, chat with other players (AI bots), and navigate between locations.

### Tech Stack
- **Phaser 3** (CDN) — game engine
- **Vanilla JavaScript** — single-file `src/game.js` architecture
- **localStorage** — account persistence, no backend server
- **Python http.server** — static file serving

### Sprite Sheet System
**Critical pattern learned:** Phaser `load.spritesheet()` with `generateFrameNumbers()` fails silently → black squares.

**Reliable approach:** Extract individual frames with Python/PIL, load as separate `scene.load.image()` assets, build animations from frame arrays.

**Sprite sheets used:**
- `test-iso_idle_down/right/up.png` — 5×5 grid, 25 frames each
- `test-iso_walk_down/right/up.png` — 5×5 grid, 25 frames each  
- `test-iso_run_down/right/up.png` — 5×5 grid, 25 frames each
- `test-iso_jump_down/right/up.png` — 5×5 grid, 25 frames each

**Extracted to:** `~/ClawMind/galaxy-demo/assets/iso/*.png` (300 total frames)

### Animation Keys
- `idle-down/right/up` — breathing loop, 3 FPS
- `walk-down/right/up` — 12-frame subset at 12 FPS
- `run-down/right/up` — full 25 frames at 18 FPS
- `jump-down/right/up` — full 25 frames, play once
- Left facing uses `setFlipX(true)` on right sprites
