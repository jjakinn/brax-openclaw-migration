# 2026-05-08 — LPC Character Integration for galaxy-demo

## What Was Built

Full LPC (Liberated Pixel Cup) character customization system integrated into the galaxy-demo Godot project.

## Architecture

### Layered Sprite System
- **LPCLayer.gd** — `class_name LPCLayer extends Sprite2D`. Displays one frame from an LPC spritesheet using `region_rect` (AtlasTexture-style). Syncs animation/direction/frame with parent animator.
- **LPCCharacterAnimator.gd** — `class_name LPCCharacterAnimator extends Node`. Drives all LPCLayer children in sync. Manages walk/idle/run animations, 4 directions (south/west/east/north), frame timing.

### Player Integration
- **Player.tscn** — Added `LPCCharacterAnimator` node with 6 pre-wired `LPCLayer` children (Body, Legs, Feet, Torso, Head, Hair).
- **Player.gd** — When `lpc_enabled`, hides the old AnimatedSprite2D and drives the LPC animator instead. No flip_h for LPC (has separate left/right frames). Textures loaded via `Image.load()` + `ImageTexture.create_from_image()` from absolute filesystem paths.

### Character Creator UI
- **CharacterCreator.tscn** — Split layout: left scrollable panel with category sections, right live preview.
- **CharacterCreator.gd** — Reads `res://assets/lpc_catalog.json` (104 categories, 655 options). UI sections: Body, Hair, Torso, Legs, Feet, Head. Click to toggle layers. Color variant pickers for items with subdirectories (e.g., frock_black, frock_blue). Live preview updates in real-time.

### Game Flow
- **LoginUI.gd** — Auto-login replaced with Continue/Create/Logout screen.
- **Main.gd** — `show_character_creator()` instantiates CharacterCreator, passes username/password.
- **AccountManager.gd** — Saves `lpc_character` dict alongside account data.

## Key Technical Decision

Instead of importing 152,000 PNGs into Godot (629MB, crashes import), textures are loaded **dynamically from absolute paths**:
```gdscript
var img = Image.new()
img.load("/Users/Jakin/ClawMind/lpc-generator/spritesheets/hair/princess/adult/fg/walk.png")
var tex = ImageTexture.create_from_image(img)
```

21 curated files in `assets/lpc/` for fast Godot startup. The full catalog (151,896 files) lives in the external LPC repo clone.

## Data Format

CharacterCreator → Player passes `lpc_spritesheets` as flat dict:
```json
{
  "body": "/Users/Jakin/ClawMind/lpc-generator/spritesheets/body/bodies/male/",
  "hair": "/Users/Jakin/ClawMind/lpc-generator/spritesheets/hair/princess/adult/fg/",
  "torso": "/Users/Jakin/ClawMind/lpc-generator/spritesheets/torso/jacket/frock/male/",
  "legs": "/Users/Jakin/ClawMind/lpc-generator/spritesheets/legs/pants2/thin/",
  "feet": "/Users/Jakin/ClawMind/lpc-generator/spritesheets/feet/boots/basic/thin/",
  "head": "/Users/Jakin/ClawMind/lpc-generator/spritesheets/head/heads/human/adult/"
}
```

Player._setup_lpc_layers() iterates `layer_order = ["body", "legs", "feet", "torso", "head", "hair"]` and creates LPCLayer nodes for each entry.

## How to Test

1. Launch game (Main.tscn)
2. If auto-logged in: click **Log Out**
3. Click **Create Account**
4. Username (4-12 chars, alphanumeric) + password
5. Character Creator appears — pick body type, browse categories, click options, pick colors
6. Click **Done** → game starts with custom layered character

## Notes

- LPC spritesheets follow 64×64 frame layout: walk=9×4, idle=1×4, run=8×4
- Directions: 0=south/down, 1=west/left, 2=east/right, 3=north/up
- Catalog file has debug line at top — CharacterCreator strips it before JSON.parse()
- `PackedStringArray.join()` does NOT exist in Godot 4 — use `"\n".join(array)` instead
