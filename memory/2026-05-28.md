## GitHub Push Pattern (Working)
```bash
# Check gh auth
git remote -v
git status

# If token needed temporarily:
git remote set-url origin https://TOKEN@github.com/OWNER/REPO.git
git push -u origin main --force  # if initial commit conflict
git remote set-url origin https://github.com/OWNER/REPO.git  # strip token
```

## Technical Notes
- GitHub Pages takes ~30-60s to propagate after push
- Browser cache aggressive on GitHub Pages — hard refresh (`Cmd+Shift+R`) or incognito required
- `sed -i ''` on macOS for in-place editing
- `sed` global replace with escaped pipes: `sed -i '' 's|old|new|g' file`
