## School Project: Forced Redirect Ad Page

### What We Built
A page that clones a real streaming site's UI (StreamEast), where clicking "Play" on a video first redirects the user to an ad URL (new tab), and on the second click actually plays the video.

### The Recipe (Works for ANY website + ANY ad)

#### Step 1: Mirror the target site with HTTrack
```bash
httrack "https://TARGET-SITE.com/page" -O ~/Desktop/site-mirror --depth=2 --near --robots=0 --timeout=30 --retries=1 -s0
```

#### Step 2: Create the forced redirect page inside the mirror
Place the HTML file **inside the mirror directory** so all relative CSS, fonts, and images work.
Example: `~/Desktop/site-mirror/TARGET-SITE.com/forced-redirect.html`

#### Step 3: Key HTML structure
- Copy the header/nav/footer from a real page in the mirror
- Reference the site's CSS file (relative path since we're inside the mirror)
- Create a fake player overlay with a play button on top of a hidden iframe
- Add the redirect JavaScript (see template below)

#### Step 4: Serve from localhost (required for YouTube embeds)
```bash
cd ~/Desktop/site-mirror && python3 -m http.server 8081
# Then open: http://localhost:8081/TARGET-SITE.com/forced-redirect.html
```
YouTube embeds throw "video configuration error" from `file://` pages — localhost fixes this.

### JavaScript Template (the core logic)
```javascript
var adPlayed = false;
var fakePlayer = document.getElementById('fakePlayer');

fakePlayer.addEventListener('click', function () {
    if (!adPlayed) {
        // FIRST CLICK: open ad/redirect in a new tab
        adPlayed = true;
        window.open('AD_URL_HERE', '_blank');
    } else {
        // SECOND CLICK: play the real video
        document.getElementById('realPlayer').src =
            'https://www.youtube.com/embed/VIDEO_ID_HERE?autoplay=1';
        fakePlayer.style.display = 'none';
    }
});
```

### HTML Player Template (the fake player + real iframe)
```html
<div class="embed-responsive embed-responsive-16by9">
    <!-- Fake player overlay (visible first) -->
    <div class="fake-player" id="fakePlayer">
        <div class="play-btn"></div>
    </div>
    <!-- Real iframe (hidden, no src until second click) -->
    <iframe id="realPlayer"
        allow="autoplay; accelerometer; encrypted-media; gyroscope; picture-in-picture"
        allowfullscreen></iframe>
</div>
```

### Key CSS for the fake player
```css
.fake-player {
    position: absolute; top: 0; left: 0;
    width: 100%; height: 100%;
    background: #000 url('THUMBNAIL_URL') center/cover no-repeat;
    display: flex; justify-content: center; align-items: center;
    cursor: pointer; z-index: 2;
}
.play-btn {
    width: 72px; height: 72px;
    background: rgba(225, 6, 1, 0.9);
    border-radius: 50%;
    display: flex; justify-content: center; align-items: center;
}
.play-btn::after {
    content: ''; border-top: 16px solid transparent;
    border-bottom: 16px solid transparent;
    border-left: 26px solid #fff; margin-left: 5px;
}
#realPlayer {
    position: absolute; top: 0; left: 0;
    width: 100%; height: 100%; border: none; z-index: 1;
}
```

### Variables to swap for any project:
| Variable | What to change |
|----------|---------------|
| `TARGET-SITE.com` | The site to clone |
| `AD_URL_HERE` | The redirect/ad URL |
| `VIDEO_ID_HERE` | YouTube video ID for the real video |
| `THUMBNAIL_URL` | YouTube thumbnail: `https://img.youtube.com/vi/VIDEO_ID/maxresdefault.jpg` |
| Play button color | Change `rgba(225, 6, 1, 0.9)` to match the site's brand |

### Gotchas
- YouTube embeds MUST be served from `http://` not `file://` (use python3 http.server)
- The iframe must exist in the DOM from page load (don't create via innerHTML — causes referrer issues)
- Set iframe `src` only on second click, not on page load
- Use `window.open(url, '_blank')` for new tab redirect (not `window.location.href` which navigates away)
